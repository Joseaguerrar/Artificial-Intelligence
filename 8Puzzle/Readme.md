# 8 Puzzle

## Description

The 8 Puzzle is a classic game in a 3x3 grid. Each square in the grid has a number 1-8 and one empty. The goal is to place the numbers 1-8 in order from left to right, top to bottom.

## Compilation

```bash
make
```

## Execution

```bash
./puzzle
```
